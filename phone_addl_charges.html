Additional Charges/Manual OMSe Orders

If customer can't call internationally and they are getting an error "you have entered a restricted code" that means international calling is Blocked in OMSe (whether or not it shows allowed in CSG).

To fix this submit an auto complete order to match CSG to OMSe (block international {and additional charges if necessary}) and complete it. Then flow a normal order to Allow international (and additional charges).

If you do not have auto complete, you will have to create a manual PRV in OMSe to fix it.

A list of all blocking options available in OMSe.

Select create, and then provisioning order to manually create a PRV

What a new PRV looks like. Ensure you select the right division for your customer's account number

Make sure you change the PRV to an Update request(the default is Activate), add the Source ID (always ACP), the account number, and Service ID (telephone number). Make sure you change the Due Date to today.
